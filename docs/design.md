# Arch Installer V2

## Goals
- UEFI only
- GPT only
- Btrfs only
- Snapper
- UKI
- systemd-boot
- Sway or Hyprland
- TTY login
- GNOME Keyring
- Seahorse
- Zsh
- NOPASSWD sudo

## Target Machines
- ThinkPad X230

## Not Supported Initially
- BIOS
- MBR
- ext4
- GRUB
- KDE
- GNOME

## Future
- LUKS encryption
- Docker profile
- Tailscale profile
